Quick Start

  1. Enter a falsifiable scientific intuition in the Input box, or upload a PDF/TXT document.
  2. Launch Cycle — ASTRA formalizes it into a hypothesis, writes a validation script, and runs it.
  3. Review the validated theorem and approve or reject adding it to the Axiomatic Base.

Troubleshooting

Cycle stuck on IDLE after clicking Launch

Ensure at least one API key is configured. Restart the wizard and run the preflight check. If running in simulated mode (no keys), the cycle will complete but results are placeholder data.

CODE_ERROR repeating on every retry

The translator is generating invalid code. Try narrowing your intuition to a more specific and concrete claim. Avoid ambiguous or overly abstract phrasing.

Server shows OFFLINE

The Flask server has stopped. Close the browser tab and relaunch the wizard from the terminal or desktop shortcut.

External CAS not found (SageMath / Maxima)

These are optional. Run install_windows_wsl.ps1 to install them via WSL. Python-only validation works without them for most use cases.

PDF upload not working

PyMuPDF must be installed. Run pip install PyMuPDF and restart the wizard. The wizard's auto-install step should handle this automatically.
