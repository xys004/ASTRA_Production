Help & Examples

Quick Start — Single Cycle

  1. Enter a falsifiable scientific intuition in the Input box, or upload a PDF/TXT document.
  2. Launch Cycle — ASTRA formalizes it into a hypothesis, writes a validation script, and runs it.
  3. Review the validated theorem and approve or reject adding it to the Axiomatic Base.

Quick Start — Research Loop

  1. Switch to Research Loop mode using the toggle at the top of the sidebar.
  2. Enter a macro question — the overarching scientific question guiding the session. You can also upload a PDF or TXT as the question.
  3. Set Heartbeat — how many cycles run between human-review pauses (not a stop, just a pause).
  4. Autonomous Mode — enable to skip all human pauses. ASTRA will run continuously and only stop when the Navigator declares the macro question resolved.
  5. Launch Session — ASTRA autonomously generates hypotheses, validates them, and navigates to the next direction.
  6. At milestones (manual mode only), review the Navigator Proposal: continue, redirect with a custom direction, or activate a parallel branch from the registry.
  7. When the session ends, the reports folder opens automatically.

Investigation Management

New — saves the current investigation and resets ASTRA for a fresh start. The Axiomatic Base and all reports are cleared.

History — browse, load, or delete past investigations. Each is saved with its cycle count and validated theorems.

The Inv. N counter in the status bar shows how many cycles have run in the current investigation (resets on New).

Thread cards show cycle results with colored badges:

CODE_ERROR — validation script failed after retries.
API_ERROR — provider API quota/error; cycle skipped, will retry.

Troubleshooting

Cycle stuck on IDLE after clicking Launch

Ensure at least one API key is configured in Settings ⚙. Keys are saved to your local .env file.

API_ERROR badge appears on every cycle

Your API quota is exhausted or the provider is down. ASTRA detects this cleanly and retries the same direction. Wait a few minutes or switch provider in Settings.

CODE_ERROR repeating on every retry

The translator is generating invalid code. Try narrowing your intuition to a more specific and concrete claim. Avoid ambiguous or overly abstract phrasing.

Server shows OFFLINE

The Flask server has stopped. Relaunch from the terminal or the desktop shortcut (launch_astra.bat).

Heartbeat vs. Autonomous Mode

Heartbeat N means ASTRA pauses every N cycles for human review — it is not a stop. In Autonomous Mode, these pauses are skipped entirely; ASTRA runs until the macro question is resolved.

Research Loop paused at WAITING_DIRECTION

A milestone was reached. The Navigator Proposal modal will appear automatically. Choose to continue, redirect with a custom direction, or activate a saved branch. Enable Autonomous Mode before launching to skip these pauses.

External CAS not found (SageMath / Maxima)

These are optional. Run install_windows_wsl.ps1 to install via WSL. Python-only validation works for most use cases.
